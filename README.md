# credit-clicker-info

Legal documents for the Credit Clicker mobile app by Remedy Point Solutions, Inc.

## Documents

- [Privacy Policy](privacy-policy.md)
- [Terms & Conditions](terms-and-conditions.md)

## Deployment

This repository will be made **public** and GitHub Pages will be enabled prior to App Store launch.

Once public, documents will be served at:
- `https://rpsapp.github.io/credit-clicker-info/privacy-policy`
- `https://rpsapp.github.io/credit-clicker-info/terms-and-conditions`

## To enable GitHub Pages (when making repo public)

1. Make the repo public: **Settings → Danger Zone → Change visibility**
2. Enable Pages: **Settings → Pages → Source: Deploy from branch → main / root**
